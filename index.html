    markdown_mermaid

    Python-Markdown extension to add Mermaid graph

    Getting Started

    Prerequisites

    • Python

    Installation

    For pip installation (only python version >=3.x) :

    pip install markdown_mermaid

    Usage

    With Python Script

    import markdown


text = """
# Title

Some text.

窶義``mermaid
graph TB
    A --> B
    B --> C
窶義``

Some other text.

窶義``mermaid
graph TB
    D --> E
    E --> F
窶義``
"""

html = markdown.markdown(text, extensions=["markdown_mermaid"])

print(html)

    Expected output

    <h1>Title</h1>
<p>Some text.</p>
<div class="mermaid">
graph TB
    A --> B
    B --> C
</div>

<p>Some other text.</p>
<div class="mermaid">
graph TB
    D --> E
    E --> F
</div>

<script type="module">
    import mermaid from 'https://cdn.jsdelivr.net/npm/mermaid@11/dist/mermaid.esm.min.mjs';
    mermaid.initialize({ startOnLoad: true });
</script>

    Use it with Pelican

    Add "markdown_mermaid": {} to MARKDOWN["extension_configs"] in your pelicanconf.py

    MARKDOWN = {
    "extension_configs": {
        "markdown_mermaid": {},
    },
}
